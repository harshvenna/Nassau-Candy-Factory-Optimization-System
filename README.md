# 🍬 SupplyChainAI — Nassau Candy Distributor

**Intelligent Factory Reallocation & Shipping Optimization Platform**

> An end-to-end Decision Intelligence system combining predictive ML, multi-objective optimization, Monte Carlo simulation, and SHAP explainability — built for academic submission at industry-grade quality.

---

## 🎯 Project Overview

Nassau Candy Distributor assigns products to factories using static legacy rules, causing:
- Suboptimal shipping distances and high lead times
- Uneven factory utilisation (Lot's O' Nuts handles 55.7% of all units)
- Margin erosion due to logistics inefficiencies
- No simulation or optimization capability

**SupplyChainAI** solves this with a complete AI-powered platform that:
1. Predicts shipping lead times with **R² = 0.999** (Gradient Boosting)
2. Predicts gross profit with **R² = 0.997**
3. Simulates factory reassignments via Monte Carlo (1,000+ runs per scenario)
4. Generates ranked reallocation recommendations using multi-objective optimization
5. Explains every prediction via SHAP (Shapley Additive Explanations)
6. Delivers insights through a 12-page professional Streamlit dashboard

## 🏗️ Architecture

```
supplychain_ai/
├── data/
│   └── candy.xlsx              # Nassau Candy dataset (10,194 orders)
├── src/
│   ├── data_pipeline.py        # Loading, cleaning, feature engineering (45 features)
│   ├── ml_models.py            # 6-model training suite + evaluation + persistence
│   ├── optimization_engine.py  # Reallocation simulation + scoring + Monte Carlo
│   └── shap_explainer.py       # Global/local SHAP explanations
├── streamlit_app/
│   └── app.py                  # 12-tab professional dashboard (1,268 lines)
├── models/                     # Persisted artefacts (auto-generated by train.py)
│   ├── artefacts.pkl           # Best ML models + encoders + metrics
│   ├── df_engineered.pkl       # Feature-engineered dataset
│   ├── shap_lt.pkl             # SHAP values for Lead Time model
│   └── shap_pr.pkl             # SHAP values for Profit model
├── outputs/                    # Analysis outputs
│   ├── recommendations.csv     # Top-100 factory reallocation recommendations
│   └── monte_carlo_summary.csv # MC simulation results for all scenarios
├── train.py                    # Master training pipeline (run once)
└── requirements.txt            # All dependencies
```

## ⚙️ Installation

### Prerequisites
- Python 3.10+
- pip

### Setup

```bash
# Clone / unzip the project
cd supplychain_ai

# Install all dependencies
pip install -r requirements.txt

# Train all models (takes ~3-5 minutes)
python train.py

# Launch the Streamlit dashboard
streamlit run streamlit_app/app.py
```

The dashboard opens at `http://localhost:8501`
